# Pizza Sales Dashboard | Power BI

## Project Overview

Developed an interactive Power BI dashboard to analyze pizza sales performance and customer purchasing behavior. The dashboard provides insights into revenue trends, product performance, sales patterns, and customer ordering habits, helping stakeholders make data-driven business decisions.

## Business Problem

Restaurant management required a centralized reporting solution to:

- Track overall sales performance
- Monitor customer ordering patterns
- Identify top and low-performing pizzas
- Analyze peak business hours
- Compare weekday and weekend sales behavior
- Support data-driven decision-making

#### Calendar Tooltip
Displayed when hovering over a specific date in the calendar visual.

- Dynamic header showing the selected date.
- Total pizzas sold on that date.
- Top-selling pizza for the selected date.
- Category-wise pizza sales breakdown by size.
- All values update based on the hovered date.

#### Weekday / Weekend Tooltip
Displayed when hovering over the Weekday vs Weekend analysis.

- Dynamic header showing either **Weekday** or **Weekend**.
- Total pizzas sold for the selected group.
- Top-selling pizza for the selected group.
- Category-wise pizza sales breakdown by size.
- All values update based on the hovered segment.

#### Date & Hour Heatmap Tooltip
Displayed when hovering over a specific date-hour combination in the sales heatmap.

- Dynamic header showing the selected day and hour.
- Total pizzas sold during that time period.
- Top-selling pizza for the selected time slot.
- Category-wise pizza sales breakdown by size.
- All values update based on the hovered date and hour.

## Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Total Revenue | Total revenue generated from pizza sales. |
| Total Orders | Total number of customer orders placed. |
| Total Pizzas Sold | Total quantity of pizzas sold. |
| Total Table Orders | Total number of dine-in table orders. |
| Average Order Value | Average revenue generated per order. |
| Average Pizzas per Table | Average number of pizzas ordered per table. |
